# Playful Interactive Portfolio — Ashish K. Shatapathy

An engaging and responsive frontend portfolio showcasing skills, projects, and creativity.

## About

This portfolio demonstrates modern frontend development using HTML, CSS, and JavaScript with interactive UI features like animated backgrounds, skill badges, playable demos, and contact integration without the need for a backend.

## Features

- Responsive design with a dark neon theme
- Cursor-reactive animated particle background
- Interactive skill cloud and HireScore meter
- Playable project demos: Tic-Tac-Toe, Rock-Paper-Scissors, Weather Dashboard
- Contact form powered by EmailJS for direct messages
- Custom badge generator for skill match visualization
- Fun bouncing ball animation in Hiring Game section

## Technologies Used

- HTML5, CSS3 (with CSS variables and media queries)
- Vanilla JavaScript (ES6+)
- EmailJS for contact forms
- Google Fonts (Inter)

## Usage

1. Clone or download the repository.
2. Open `index.html` in a modern web browser.
3. Interact with skill badges, project demos, and contact form.

## How to Configure Contact Form

- Create an account on [EmailJS](https://www.emailjs.com/).
- Set up your email service and template.
- Replace `'YOUR_USER_ID'`, `'YOUR_SERVICE_ID'`, and `'YOUR_TEMPLATE_ID'` in `script.js` with your EmailJS credentials.
